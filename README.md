# 🚀 Jenkins Server Deployment with Terraform on AWS

This project automates the creation of a Jenkins server on **AWS** using Infrastructure as Code (IaC) principles.

## 📋 Prerequisites
* **Terraform** installed on your local machine.
* An active **AWS Account** with configured credentials.
* **Git** for version control.

## 🛠️ Usage
1. **Clone** the repository to your local machine.
2. Run `terraform init` to initialize the working directory and download providers.
3. Run `terraform plan` to see the infrastructure changes.
4. Run `terraform apply` to deploy the Jenkins server on AWS.

## 🛡️ Infrastructure Details
* **Cloud Provider:** AWS
* **Service:** EC2 Instance
* **Tooling:** Terraform & Jenkins
